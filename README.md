# Doom Scrolling

A TikTok-style educational scrolling prototype that turns screen time into learning.

Live demo: https://hemanth2665.github.io/doom-scrolling/

## Prototype features

- YouTube Shorts embedded inside the Doom Scrolling feed
- AI, cybersecurity, cloud, coding, career, and learning categories
- Like, comment, save, quiz, upload, premium, analytics, profile, and DM screens
- Theme switcher including White + Blue and Ionized Purple + Blue
- Mobile-first browser experience

## Note

The prototype uses the official YouTube embed player inside the app. Some YouTube creators or videos may block embedding. If a video is blocked by YouTube, replace it with a different embeddable video or use an owned/licensed MP4 file in `assets/videos/`.
