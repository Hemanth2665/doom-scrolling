Doom Scrolling - Pitch Ready Prototype V5

Open index.html in a browser.

What is inside:
- Startup landing page plus clickable mobile app prototype.
- Real educational YouTube Shorts embedded in the feed.
- Working app screens: Home, Profile, DM, Upload, Premium, Analytics, Search.
- Theme Studio with White + Blue, Ionized Purple + Blue, Cobalt Pro, Aurora Neon, Graphite Blue, and custom colors.

- Some browsers may block autoplay until the user interacts with the page. The Original button opens the source video.

- This is a static frontend prototype. No backend is required yet.
